Mobile Installation Considerations
(cont’d)
n A compact 12V mobile TNC provides both APRS
send and receive capability. e.g. TNC-X
http://www.tnc-x.com/
n Tracker function or digipeat function (not both)
can be added to TNC-X.
n However, unless text message send/receive
and/or map display functionality is required, a TNC
is not needed.
n Tiny-Trak3 is the most compact and cheapest
tracker solution at around US$50 plus handheld
and 2M antenna. http://www.byonics.com/
