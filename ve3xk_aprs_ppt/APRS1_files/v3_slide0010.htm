AX.25 Amateur Packet Protocol
Transmissions contain small blocks of information
called frames
n Information (I) Frames – contain the user data
being transferred across the network
n Supervisory (S) Frames – provide control of the
communications link e.g. acknowledging receipt or
requesting resend
n Unnumbered (U) Frames provide additional
control, and make it possible to send Unconnected
(UI) Frames used for calling CQ, and sending
announcements (beacons)
