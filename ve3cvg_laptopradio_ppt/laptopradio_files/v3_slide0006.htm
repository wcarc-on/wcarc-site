Amateur Radio Opportunities
• A frequency can be selected which falls at least
22MHz below the top of the amateur radio
2.4GHz band. i.e.Channels 1 to 4. Probably 5 is ok
• A network can be set up using an Access Point
and one or more clients (computers)
• Access Point functions like a digipeater
• Two clients may communicate directly on a peer-
to-peer basis
