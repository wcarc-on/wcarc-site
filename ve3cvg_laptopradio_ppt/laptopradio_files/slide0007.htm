Amateur Radio peer-to-peer #1
•Requires changing hardware settings for the 802.11b wireless card.
•For this to work you will need a common SSID (use same ham callsign for both stations), a common frequency, and an application
•Freeware applications are available which permit text messaging (see IC3), VoIP (see LAN ic), file sharing.
•Before you change settings, write them down so that you can restore normal operation later.
•Range depends on Tx power out, antenna, and Rx sensitivity
